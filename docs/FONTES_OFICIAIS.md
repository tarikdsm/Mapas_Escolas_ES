# Fontes Oficiais

## Escolas estaduais

- Base georreferenciada consolidada do projeto de escolas do Espírito Santo
- Origem primária: Governo do Estado do Espírito Santo
- Validação complementar em parte da base: INEP

## Densidade populacional

Fonte oficial usada na camada municipal do site:

- IBGE Cidades e Estados
- API de Malhas do IBGE

Links:

- https://www.ibge.gov.br/cidades-e-estados/es/
- https://servicodados.ibge.gov.br/api/docs/malhas?versao=3
- https://servicodados.ibge.gov.br/api/v1/localidades/estados/32/municipios

## Observação de escopo

Nesta primeira entrega, a camada de densidade foi configurada para o Espírito Santo, porque todo o mapa e a base escolar do repositório estão centrados no ES. A estrutura dos scripts foi deixada preparada para adaptação futura caso seja necessário publicar outra UF.
